# D4JSP-Map

Diablo 4 Sanctuary interactive map. Vite + Leaflet + Fuse.js. Iframed inside the D4JSP trade site's Profile tab.

**Read [`start.md`](./start.md) first.** Everything else is reachable from there.

This app has no Supabase, no auth, no framework — just static map tiles + Leaflet overlays + a localStorage-backed build planner modal.
